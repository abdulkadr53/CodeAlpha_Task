Dormitory Management System

This system offers a comprehensive solution for managing student accommodations within academic institutions.

It manages the entire lifecycle of dormitory operations including registration, bed assignment, fee tracking, maintenance requests, and transfers.

Key features:

  • Student and address management with emergency contacts
  • Real-time bed and room assignment tracking
  • Maintenance request logging and auto-status updates
  • Fee tracking with automatic payment updates
  • Locker assignment and room transfer history
  • Advanced querying: check available rooms, unpaid fees, and more

It uses role-based access control, ensuring that students, proctors, and admins access only relevant functions.

The system is scalable and can be integrated with both web and desktop interfaces, making it ready for real-world deployment.
